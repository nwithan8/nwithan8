### Peace out, GitHub.

Microsoft has proven itself to be completely incapable of operating GitHub, both through gross violations of personal data privacy and copyright and (likely related due to the vibe-coding they push by shoving AI down everyone's throats) literally just being unable to keep the site up and functioning.

I am in the process of migrating over to [Codeberg](https://codeberg.org/nwithan8). Migration strategy:
- Any *actively-maintained* public repo on GitHub has been migrated to Codeberg effective immediately. The GitHub repo has had been archived, with all branches and commits stripped, and a simple redirect on the README to the corresponding Codeberg repo.
  - These landing pages are subject to future removal.
- Any stale or abandoned projects have been moved off-site for archival purposes. The GitHub repo has been deleted.
  - If you require something from one of these old repos, please reach out to me.
- Any private repos have been migrated to the appropriate new home. The GitHub repo has been deleted.

A few repos require their presence on GitHub to work in various systems. These repos are read-only mirrors of the active development on Codeberg. While they may appear "active", please do not open any PRs or issues on these repos, as they will be ignored. Direct all activity to the corresponding Codeberg repository.
